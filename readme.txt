This GO Application will use GIN for it webAPI

Using GIN for Go

** Dependencies:
install
go get github.com/gin-gonic/gin


** Build Instruction:
To build project: cmd execute build.bat

** Execution Instruction:
To run : cmd execute RestFulAPI-test.exe

To check website add the following to the address bar of your browser.
http://localhost:8080/api/v1/healthcheck

** Testing:
To test this application I would use Postman application.


** Risk involved this application:

This simplae application can open itself up to security risks as the application run on http://. A more secure way would to load this application to a website with ssh encription and run as https://
